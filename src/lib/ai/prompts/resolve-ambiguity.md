# 消除歧义 Prompt

## 任务
当AI需要更多信息时，向用户提出有针对性的问题

## 输入
- 用户的原始想法
- AI的理解和疑问
- 上下文信息

## 输出格式
```json
{
  "question": "需要用户回答的问题",
  "options": ["选项1", "选项2", "选项3"],
  "reasoning": "为什么需要这个问题"
}
```

## Prompt
```
你是一个写作教练，正在与用户讨论他们的写作想法。

用户说：{{originalIdea}}

{{#if context}}
上下文：{{context}}
{{/if}}

你需要理解用户的意思，但有些地方不够清晰。请提出1~2个关键问题来澄清：

要求：
1. 问题要具体、有针对性
2. 提供清晰的选项让用户选择
3. 问题要帮助明确写作方向或范围
4. 不要问过于宽泛或开放的问题

请以JSON格式输出：
{
  "questions": [
    {
      "question": "具体问题",
      "options": ["选项1", "选项2", "选项3", "其他（请说明）"],
      "reasoning": "为什么问这个问题"
    }
  ]
}
```

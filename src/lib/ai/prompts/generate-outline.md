# 生成大纲 Prompt

## 任务
根据核心论点生成完整的文章大纲

## 输入
- 核心论点列表
- 可选：文章标题

## 输出格式
```json
{
  "title": "文章标题",
  "outline": [
    {
      "id": "1",
      "title": "章节标题",
      "description": "章节描述/要点",
      "children": [
        {
          "id": "1-1",
          "title": "子章节标题",
          "description": "子章节描述"
        }
      ]
    }
  ]
}
```

## Prompt
```
你是一个专业的博客写作专家，擅长构建清晰、有逻辑的文章结构。

请根据以下核心论点，生成一篇完整的博客文章大纲：

核心论点：
{{corePoints}}

{{#if title}}
参考标题：{{title}}
{{/if}}

要求：
1. 大纲应包含引言、3~5个主要章节、结论
2. 每个章节需要有清晰的标题和描述
3. 章节之间要有逻辑递进关系
4. 结构要适合博客文章格式（不宜过于学术化）

请以JSON格式输出：
{
  "title": "文章标题",
  "outline": [
    {
      "id": "1",
      "title": "章节标题",
      "description": "本章要讲的核心内容",
      "children": [
        {
          "id": "1-1",
          "title": "子章节标题",
          "description": "子章节要点"
        }
      ]
    }
  ]
}
```

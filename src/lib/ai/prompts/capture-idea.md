# 捕捉想法 Prompt

## 任务
将用户模糊的想法拆解成3~5个核心论点，帮助明确写作方向。

## 输入
用户的原始想法（可能很模糊、零散）

## 输出格式
```json
{
  "core_points": ["论点1", "论点2", "论点3", ...],
  "summary": "对用户想法的简短总结和理解"
}
```

## Prompt
```
你是一个专业的写作教练，擅长将模糊的想法转化为清晰的写作方向。

用户输入了一个写作想法，可能比较模糊或零散。你的任务是：
1. 理解用户的核心意图
2. 将其拆解成3~5个核心论点
3. 帮助用户明确写作方向

用户想法：
{{idea}}

请以JSON格式输出，格式如下：
{
  "core_points": ["论点1", "论点2", "论点3"],
  "summary": "对这个想法的理解和写作方向的建议"
}
```

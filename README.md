## ✨ 技术栈

- **框架**: Next.js 16 (App Router)
- **UI**: shadcn/ui + Tailwind CSS v4
- **语言**: TypeScript
- **部署**: GitHub Pages

## 🚀 本地运行

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

# 截图说明

## 需要添加的截图

请在运行 Streamlit 项目后，将以下截图保存到此目录：

1. **initial-page.png** - 应用初始页面（显示标题和输入框）
2. **chat-demo.png** - 对话效果页面（显示多轮对话）

## 如何获取截图

1. 安装依赖并运行应用：
   ```bash
   pip install streamlit openai
   streamlit run app.py
   ```

2. 在浏览器中截图：
   - 初始页面截图保存为 `initial-page.png`
   - 对话页面截图保存为 `chat-demo.png`

3. 将截图文件复制到此目录：
   ```bash
   cp path/to/initial-page.png public/blog/streamlit-ai-companion/
   cp path/to/chat-demo.png public/blog/streamlit-ai-companion/
   ```
